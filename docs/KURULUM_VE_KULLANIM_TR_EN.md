# Kurulum ve Kullanım / Installation and Usage

## TR - Hızlı Kurulum

1. Python 3.10 veya 3.11 kurun.
2. Intel RealSense SDK ve kamera sürücülerini kurun.
3. Sanal ortam oluşturun.
4. `pip install -r requirements.txt` komutunu çalıştırın.
5. Arduino kodunu Arduino Uno'ya yükleyin.
6. `komutlar/baslat_stabil_yuz.bat` dosyasını çalıştırın veya README'deki komutu kullanın.

## EN - Quick Setup

1. Install Python 3.10 or 3.11.
2. Install Intel RealSense SDK and camera drivers.
3. Create a virtual environment.
4. Run `pip install -r requirements.txt`.
5. Upload the Arduino firmware to Arduino Uno.
6. Run `komutlar/baslat_stabil_yuz.bat` or use the command in README.
